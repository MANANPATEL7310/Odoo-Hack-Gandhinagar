# 🧠 Local Planning Directory

**STOP! Do not write code yet.**

This team requires a **plan-first workflow** to reduce bugs, avoid AI hallucinations, and ensure production-ready code. 

## The Workflow:
1. When building a new feature, ask your AI (Cursor, Gemini, Copilot, etc.) to generate an `implementation_plan.md` in this `.planning/` directory.
2. Review the plan yourself. Adjust the database schema, API routes, or component designs as needed.
3. Once the plan looks perfect, give the AI the green light to execute the code.

*Note: The contents of this folder (except this README) are git-ignored. Your planning files remain on your local device and will not clutter the GitHub repository.*
